# WEB103 Project 1 - *Festival Lineup Explorer*

Submitted by: **Ying Wang**

About this web app: **Festival Lineup Explorer is a web application that allows users to browse festival artists and view detailed information about each performer, including their set times, stages, genres, and images. Users can navigate between artist detail pages through unique URLs and receive a custom 404 page for invalid routes.**

Time spent: **8 hours**

## Required Features

The following **required** functionality is completed:

* [x] **The web app uses only HTML, CSS, and JavaScript without a frontend framework**
* [x] **The web app displays a title**
* [x] **The web app displays at least five unique list items, each with at least three displayed attributes (such as title, text, and image)**
* [x] **The user can click on each item in the list to see a detailed view of it, including all database fields**

  * [x] **Each detail view uses a unique endpoint**
  * [x] **Unique URLs are demonstrated in the video walkthrough**
* [x] **The web app serves an appropriate 404 page when no matching route is defined**
* [x] **The web app is styled using PicoCSS**

## Optional Features

The following **optional** features are implemented:

* [x] The web app displays items using card-based layouts instead of standard lists.

## Additional Features

The following **additional** features are implemented:

* Responsive design for desktop and mobile devices.
* Custom navigation bar.
* Hover effects on artist cards.
* Improved error handling and user-friendly messages.
